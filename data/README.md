# Data Directory

This directory is for local data storage. Data files are NOT committed to git.

## Data Storage Locations

| Data | Location | Size Estimate |
|------|----------|---------------|
| Raw collected papers (JSONL) | `raw/` | ~5-10 GB |
| LanceDB vector database | `lancedb/` | ~20-50 GB |
| Training data (JSONL) | `training/` | ~500 MB |
| Evaluation results | `evaluation/` | ~50 MB |

## Data Access

- Raw paper data is collected via Kestra flows and stored on the Kestra server
- LanceDB data is stored at `/data/lancedb` on the server (mounted volume)
- Training data is generated via the `generate-training-data` Kestra flow
- All data can be regenerated by running the collection and processing flows

## Reproducibility

All data is reproducible from the Kestra flows in this repository. To regenerate:

1. Deploy Kestra (see `docs/kestra-setup.md`)
2. Import collection flows
3. Run `collect-openalex` for each domain
4. Run `chunk-and-embed` processing flow
5. Run `generate-training-data` for each skill type

## Privacy

- No personally identifiable information is stored
- All source papers are openly licensed
- No proprietary data is used
