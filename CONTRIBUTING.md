# Contributing to OLS Research System

Thank you for your interest in contributing to open science infrastructure. This project is part of Optimal Living Systems, a mutual aid nonprofit.

## How to Contribute

### Researchers
- Suggest additional data sources for our four domains
- Review training data examples for accuracy
- Propose new research questions for evaluation
- Validate methodology critique outputs

### Data Scientists / ML Engineers
- Improve Kestra workflow efficiency
- Optimize RAG retrieval quality
- Contribute evaluation benchmarks
- Improve chunking and embedding strategies

### Information Architects / Librarians
- Improve taxonomies and controlled vocabularies
- Suggest better paper quality metrics
- Help design the cross-domain concept index
- Review metadata schemas

### General Contributors
- Improve documentation
- Report bugs in Kestra flows
- Test the system with real research questions
- Translate documentation

## Development Setup

1. Fork the repository
2. Clone your fork
3. Follow setup instructions in `docs/kestra-setup.md`
4. Create a branch for your changes
5. Submit a pull request

## Code of Conduct

We follow mutual aid principles: cooperation over competition, community benefit over individual gain, transparency over secrecy. Be kind, be constructive, be open.

## Questions?

Open an issue or email research@optimallivingsystems.org
